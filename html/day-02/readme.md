# Day 02 — Registration Form

Practiced building a basic registration form using plain HTML — no CSS or JavaScript yet. The goal was to get comfortable with form structure, input types, and connecting labels to inputs properly.

## What the code does

- **Name fields** — two text inputs (first name, last name), each with a placeholder
- **Gender select** — a dropdown (`<select>`) with three options: Male, Female, Other
- **Terms checkbox** — a checkbox the user must tick to agree to terms and conditions, paired with a `<label>` linked via `for`/`id`
- **Submit button** — a `Register` button that submits the form to `/action`
- Related fields are grouped inside `<div>` containers for structure

## Concepts practiced

- Form structure with the `<form>` element and an `action` attribute
- Different input types: `text` and `checkbox`
- The `<select>` / `<option>` dropdown pattern
- Linking labels to inputs using `for` and matching `id` attributes (for accessibility)
- Grouping related fields inside `<div>` containers
