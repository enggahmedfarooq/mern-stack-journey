#  MERN Stack Journey

This repository documents my MERN stack web development journey — covering HTML, CSS, JavaScript, React, Node.js, Express, and MongoDB as I build toward full-stack proficiency.

I'm a Software Engineering student at UET Mardan, building this repo as both a personal learning log and a public record of my progress as I move from full-stack web development toward AI/ML later in my degree.

## 📌 About This Repo

Each topic gets its own folder. If a topic takes more than one day to complete, it's broken down further into day-based subfolders. Every folder contains:
- The code I wrote
- A `README.md` explaining what I learned, the problems I ran into, and how I solved them

This isn't just a code dump — it's meant to show *how* I think through problems, not just the final solution.
