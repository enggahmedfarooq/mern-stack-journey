# Day 03 — Styling with CSS

Practiced applying basic CSS to plain HTML elements — selectors, colors, typography, and text styling properties. The goal was to get comfortable with how CSS attaches to HTML via a linked stylesheet and how different text/font properties affect appearance.

## What the code does

- **Heading (`h2`)** — colored orange to stand out as an intro line
- **Paragraph (`p`)** — white text on a black background for contrast
- **Button** — white text on a blue background
- **Main heading (`h1`)** — a large, centered, uppercase heading styled with a custom font, bold weight, and a teal double underline
- HTML and CSS are connected via a separate `style.css` file linked in the `<head>`

## Concepts practiced

- Linking an external stylesheet using `<link rel="stylesheet">`
- Element selectors (`h1`, `h2`, `p`, `button`)
- Text color with `color` and background color with `background-color`
- Font styling: `font-family`, `font-size`, `font-weight`
- Text alignment with `text-align`
- Line spacing with `line-height`
- Text transformation with `text-transform: uppercase`
- Custom `text-decoration` with line type, style, and color (underline, double, teal)
